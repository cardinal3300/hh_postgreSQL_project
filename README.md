# 📊 HeadHunter Project (PostgreSQL)

Проект для сбора информации о работодателях и вакансиях с сайта [hh.ru](https://hh.ru), сохранения данных в PostgreSQL и работы с ними через Python.

---

## 🛠 Стек технологий:
- БД: PostgreSQL (проектирование схем, связей и ограничений).
- Язык: Python 3.12.10.
- Библиотеки: psycopg2 (работа с БД на низком уровне), requests.
- Инструменты: Интеграция с конфиг-файлами (database.ini) для безопасного хранения доступов.

---

## Описание

Программа позволяет:

- Создавать базу данных и таблицы для работодателей и вакансий.
- Загружать данные о работодателях и их вакансиях с HH API.
- Сохранять данные в PostgreSQL для последующей работы.
- Получать статистику и фильтровать вакансии по различным критериям.
- Взаимодействовать с пользователем через текстовый интерфейс.
---

## 🚀 Технические особенности:
1. **Проектирование схемы БД:** Создана нормализованная структура из таблиц `companies` и `vacancies` с использованием Foreign Keys.
2. **Класс DBManager:** Реализован интерфейс для аналитики, включающий методы:

    - `get_companies_and_vacancies_count()` — агрегация данных.
    - `get_all_vacancies()` — выборка с JOIN.
    - `get_avg_salary()` — расчет средних показателей.
    - `get_vacancies_with_higher_salary` - получение всех вакансий с з/п выше средней.
    - `get_vacancies_with_keyword()` — гибкий поиск по базе.
3. **Безопасность:** Использование параметров в SQL-запросах для защиты от **SQL-инъекций**.
4. **Автоматизация:** Скрипт сам создает базу и таблицы «с нуля», если их нет.

    ├─ main.py # Основной скрипт для запуска
    ├─ README.md # Этот файл
    └─  pyproject.toml

## Установка и настройка

1. Клонируйте репозиторий:

```bash
    git clone git@github.com:cardinal3300/hh_postgreSQL_project.git
    cd hh_postgreSQL_project
```

2. Создайте виртуальное окружение и установите зависимости через Poetry:

```bash
    poetry install
    poetry shell
```

3. Создайте файл `.env` с параметрами БД:
```
DB_NAME=
DB_USER=
DB_PASSWORD=
DB_HOST=
DB_PORT=
```

## Использование

1. Запуск основного скрипта:

```bash
    python main.py
```

2. Программа выполнит:

- Создание базы данных и таблиц.

- Получение данных о работодателях и вакансиях.

- Заполнение таблиц.

- Демонстрационный поиск вакансий по ключевому слову.

- Запуск интерфейса пользователя.
